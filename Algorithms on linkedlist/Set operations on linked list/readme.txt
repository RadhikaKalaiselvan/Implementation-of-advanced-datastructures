Given two linked lists implementing sorted sets, write functions for union, intersection, and set difference of the sets

How to run?
javac ListUtil.java
java ListUtil
then enter no. of nodes in list 1,
enter the elements in list 1
enter no. of nodes in list 2,
enter the elements in list 2

1
2

4

3

5
6
